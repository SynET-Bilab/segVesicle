# Keyboard Shortcuts Documentation

## Description
This document lists the keyboard shortcuts available for adding and deleting labels, as well as centering the cross in the application.

## Shortcuts

### Add Labels
- **Add 3D Label**: `g`
  - Use this shortcut to add a 3D label.
- **Add 2D Label**: `f`
  - Use this shortcut to add a 2D label.
- **Add 6pts Label**: `h`
  - Use this shortcut to add a 6pts label.

### Delete Label
- **Delete Label**: `d`
  - Use this shortcut to delete labels.

### Center Cross
- **Center Cross**: `c`
  - Use this shortcut to center the cross.
